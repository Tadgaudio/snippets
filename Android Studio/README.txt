Android Studio confs.
===============================

Meus live templates e demais configurações utilizadas dentro do android studio.

Para pode configura corretamente, basta ir nas configurações do editor Android Studio.
Pesquisar por live templates
Criar uma seção( caso queira ) e adicionar os live templates. Para que eles funcionem
eu costumo configurar para executar com apertando a tecla TAB apos digitar a palavra que ativa o
live template, depois de escrever a ação do live template clique no link abaixo da caixa de bloco
e defina um contexto onde esse bloco vai ser executado, clique em aplicar e pronto. Quando estiver
no contexto da linguagem definida basta digitar o nome do live template e apertar a tecla TAB.
